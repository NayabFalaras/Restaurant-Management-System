# 📖 Project Overview

The **Restaurant Management System (RMS)** is a console-based application developed using **C#** and **.NET Framework**. The system is designed to simplify restaurant operations by managing customers, menu items, orders, staff workflows, and file-based data persistence.

This project provides an efficient and organized way to handle restaurant activities through a role-based system that supports **Admin**, **Staff**, and **Customer** operations.

The application demonstrates concepts of:

* Object-Oriented Programming (OOP)
* File Handling
* CRUD Operations
* Authentication & Authorization
* Console-Based User Interface
* Data Persistence using Text Files

---

# 🌟 Features

## 👤 Multi-Role User System

### 🔐 Admin Panel

* Manage restaurant menu items
* Add, update, or delete food products
* View complete order history
* Monitor customer feedback
* Manage staff activities

### 👨‍🍳 Staff Dashboard

* View incoming customer orders
* Update order status
* Track preparation progress
* Generate daily sales reports

### 🧑‍🍽️ Customer Portal

* Browse menu categories
* Place orders
* View bill generation
* Submit feedback
* Track order status

# ⚙️ Core Functionalities

* File-based database system using `.txt` files
* Persistent storage without SQL database
* Input validation and error handling
* Console UI for easy interaction
* Order lifecycle management
* Authentication system with role-based access
* Cart-based ordering workflow

# 🚀 Installation Guide

## 📋 Prerequisites

Before running the project, ensure you have:

* Windows 10 or later
* Visual Studio 2022
* .NET Framework Installed

---

## ⚡ Steps to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/NayabFalaras/Restaurant-Management-System.git
```

### 2️⃣ Open Project

* Open **Visual Studio 2022**
* Load the `.sln` project file

### 3️⃣ Build and Run

* Press **Ctrl + F5**
* Launch the Console Application
